Desktop Applications
This is a fun little project that I wrote for the .NET Framework in C#. The program is simply a console applicaton that generates a pseudo-random password when given a keyword. I say pseudo-random because the application does not simply give you back a mess of completely random characters, but rather, a compilation of real words strung together after some manipulation. The keyword given will be used as one of the words in the password 100% of the time. APECA literally means "A Private Encrypted Chat Application." APECA runs on a client-server model with completely end-to-end encryption. All messages between clients are encrypted with 128bit AES (with CBC), and the key must be input manually by each user. The encryption key is never transmitted over the network, and the server never has the key. I made this desktop application with C#/.NET primarily because I wanted to learn about the encryption libraries available in the .NET Framework. This project also gave me the opportunity to learn a lot about low-level networking and process threading. Init is a project templating application.
